# 中文学术文本自然化改写 Skill

这是一个用于 Codex 的中文学术写作 Skill，适合处理论文、课程作业、报告、案例分析、文献综述等文本。它既可以用于论文写作过程中，帮助生成更自然、更成熟的学术表达，也可以用于后期降重改写，减少重复句式、模板化转折和空泛判断。

## 适用场景

- 降低文本中的机器腔和模板感
- 论文写作过程中生成自然、稳妥的学术段落
- 对中文学术段落进行自然化润色
- 在保留原意的前提下降重改写
- 减少重复表达、重复句式和套话
- 让段落分析更具体、更有落点

## 不适用场景

- 承诺通过任何 AI 检测或查重系统
- 隐瞒抄袭、删除必要引用或伪造原创性
- 编造文献、数据、案例或事实
- 将他人观点改写成没有来源的作者观点

## 安装方式

将本文件夹复制到 Codex 的 skills 目录中：

```text
C:\Users\你的用户名\.codex\skills\academic-naturalizer-skill
```

确保目录中包含：

```text
academic-naturalizer-skill/
  SKILL.md
  README.md
  examples/
```

之后在 Codex 中可以这样使用：

```text
请使用 academic-naturalizer-skill 帮我把下面这段论文文字改得自然一些，并适当降重。
```

## GitHub 发布建议

公开发布时，建议使用如下仓库描述：

```text
A Codex skill for naturalizing Chinese academic writing while preserving meaning, citations, and authorial intent.
```

中文描述可以写：

```text
用于中文论文写作、自然化润色和后期降重改写的 Codex Skill，强调保留原意、减少模板化表达、改善段落具体性。
```

不建议在仓库标题或简介中使用“绕过 AI 检测”“AI 查重必过”等表述。
